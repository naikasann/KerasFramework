# KerasFramework

Kerasを使うときのフレームワーク

--------------

## 概要

Kerasを用いて学習するときに何度もコードを書くのがもったいないと感じたため，
ある程度なんでも対応できるように作成すれば手間かからないなあと感じたためフレームワークのような形で作成しようと考えた．

説明は各フォルダー内にREADMEを用意しておく

--------------

## ファイル構成

ファイル構成は以下の通り

- /train --- 学習用のプログラムが入っているもの
- /tools --- 学習する際の前処理的な部分が入っている(きれいには書かないかも...)
